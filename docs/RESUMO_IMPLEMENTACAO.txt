🔧 FUNÇÕES UTILITÁRIAS CRIADAS

script.js (disponível em todos páginas)
───────────────────────────────────────

obterCanalAtendimento(atendId)
├─ Retorna: canal do atendimento
├─ Entrada: ID do atendimento
└─ Usa: localStorage obterCanalAtendimento

obterEmoji(canal)
├─ Retorna: emoji correspondente ao canal
├─ Entrada: nome do canal
└─ Mapa: CANAL_EMOJIS object

formatarCanal(canal)
├─ Retorna: emoji + espaço + canal
├─ Entrada: nome do canal
└─ Exemplo: "☎️ Telefone"

demo.html (parsing e importação)
─────────────────────────────────

importarArquivoTxt(event)
├─ Ação: Lê arquivo TXT fornecido
├─ Parser: Extrai exemplos
└─ Resultado: Popula localStorage

extrairExemplos(texto)
├─ Ação: Divide texto por delimitadores
├─ Busca: Padrão "EXEMPLO N"
└─ Retorna: Array de exemplos

parsearExemplo(bloco)
├─ Ação: Regex para campos individuais
├─ Mapeamento: Para formato interno
└─ Retorna: Objeto formatado

criarAtendimentoDoExemplo(exemplo)
├─ Ação: Cria atendimento em localStorage
├─ Dados: Atendimento + Matriz
└─ Resultado: Registros salvos
