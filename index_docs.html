✨ Funcionalidades Principais

✅ Registro Completo

Captura dados do relator, fato, classificação e relato para documentação oficial

📊 Matriz de Risco

6 eixos de avaliação com cálculo automático (0-18 pts) e classificação

🔀 Setores Dinâmicos

Dropdown com USI, Gepes, Especializadas e Superintendências. Responde conforme necessário

📝 Formulário Inteligente

Pré-preenchido com dados do atendimento + campo para parecer + redirecionamento automático

⏱️ Prazos Dinâmicos

Calculados automaticamente em dias (1/3/5/7) conforme setor e risco

📋 Fila Integrada

Todos os atendimentos com status calculado em tempo real. Sincroniza a cada 30s

💾 Offline-First

Funciona 100% sem internet. localStorage persiste dados indefinidamente

🎨 UI Responsiva

Interface profissional que funciona em desktop, tablet e mobile

💡 Dicas de Uso

🚀 Para Começar Agora

Abra demo.html e clique em "Criar Demo: Risco ALTO". O sistema pré-preencherá dados e você poderá seguir o fluxo completo.

📊 Para Entender a Integração

Leia FLUXO_INTEGRADO.md para ver diagrama completo de como os dados fluem entre as páginas.

🧪 Para Testar Completamente

Siga o GUIDE_DE_TESTE.md com 8 testes passo-a-passo que cobrem todos os cenários.

🐛 Para Debugar

Abra Console (F12) e visualize localStorage com: JSON.parse(localStorage.getItem('atendimento_AT-...'))
